# Sarang Dimsum — demo site

Demo/spec marketing site for **Sarang Dimsum** — halal dimsum café in Jalan Ciptaland, Tiban, Batam. Built to pitch the owner; not a paid engagement.

Bahasa-first, mobile-first, illustration-led. 6 pages: Home, Menu, Lokasi & Jam, Sarangers, Sewa Steling, Tentang. Astro + Tailwind v4 static build, target <1MB, LCP <2.5s on 4G.

## Run

```bash
npm install
npm run dev
```

## Notes

- The chef character on the logo is **Arfan**. Catchphrase: *Mampir, yuk.*
- Palette: cream 60 / charcoal 30 / Sarang yellow 10.
- Menu items, Sewa Steling rental terms, and reseller pipeline are **placeholder** content marked `// TODO(owner)` — owner to confirm before going live.
- All consumer-facing copy is Bahasa Indonesia. English is deferred.
